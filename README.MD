# GO BOOK SOME RASSLIN
This is a project inspired by the good ole days of me e-fedding. I partially booked some of the feds, managing rosters, reviewing promos, booking/writing shows, assigning champions, shuffling rosters. I was feeling nostalgic about those days and imagining that I would have a good system to assist me and thought, hey? Maybe

Each features here are essentially sub-projects that I will focus on. They'll take months or probably even years as I'm still a beginner/jr, I honestly just want a project I can start simple but dive deeper into. Also a fun way for me to learn Go!

# FEATURES
- Main Focus: Roster Management System
- Promo Submission
- Promo Review and Catalogue
- Booking Shows
